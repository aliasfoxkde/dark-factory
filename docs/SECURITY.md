# Security

## Reporting Vulnerabilities

Please report vulnerabilities responsibly — do not open public issues.

### Private Reporting (Preferred)
1. Go to **Security** tab → **Advisories** → **Report a vulnerability**
2. Or email: `github@aliasfoxkde.com`

### Response Timeline
- **72 hours:** We acknowledge receipt
- **7 days:** Initial severity assessment
- **30 days:** Resolution or proposed timeline

### What We Care About
- False negatives in security patterns
- ReDoS in pattern matching
- Malicious pattern injection
- Secrets leakage
- Denial of service via crafted inputs

### Out of Scope
- Social engineering
- Physical security
- DDoS on our infrastructure
